# **App Name**: ProfileGrid

## Core Features:

- Profile Display: Showcase key personal and professional information.
- Key Metrics: Highlight important achievements and performance indicators.
- Navigation: Provide links to different sections: Story, Projects, and Community.
- Tech Stack: Display the technologies and tools I am proficient in.
- Contact Information: Provide ways for visitors to get in touch (email, social media).
- Data Fetching: Fetch content from a Supabase database (or a local JSON file)

## Style Guidelines:

- Dark background (#121212) for a modern SaaS dashboard look.
- Primary color: Saturated violet (#9F00FF) for energy and tech vibe.
- Accent color: Bright pink (#FF69B4) to draw attention to key elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headers and 'Inter' (sans-serif) for body text to give a modern, readable feel.
- CSS Grid layout ensuring all 5 cards fit within 100vh on desktop without scrolling.
- Use Tailwind CSS utility classes for spacing, sizing, and responsive design.
- Subtle transitions for card hover effects or navigation changes.
